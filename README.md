## Railway ga deploy qilish

1. [railway.app](https://railway.app) ga kiring
2. **New Project** → **Deploy from GitHub repo** bosing
3. Bu fayllarni GitHub ga yuklang va repo ni ulang
4. **Variables** bo'limiga quyidagilarni qo'shing:

| Variable | Value |
|----------|-------|
| `BOT_TOKEN` | `8817431816:AAEZcjifgPtSUbm-8scMeRhZHyQMsg_ViyY` |
| `ADMIN_ID` | `7948989650` |

5. Deploy avtomatik boshlanadi ✅
